# Student Management System

A web-based Student Management System built with Django that allows users to manage student records through a simple and responsive interface.

## Overview

This project was developed to practice Django fundamentals and implement a complete CRUD (Create, Read, Update, Delete) workflow for managing student information.

Users can create, edit, delete, and view student records through a Bootstrap-based interface.

This project was originally developed in early 2025 as part of my journey learning Django and web development.

## Features

- Add new students
- Edit existing student information
- Delete student records
- View all registered students
- Responsive user interface using Bootstrap

## Student Information

Each student record includes:

- Student ID
- Full Name
- Email Address
- Major
- GPA

## Technologies Used

- Python
- Django
- Bootstrap
- SQLite

## Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/student_management_system.git
```

2. Navigate to the project directory:

```bash
cd student_management_system
```

3. Install dependencies:

```bash
pipenv install
```

4. Activate the virtual environment:

```bash
pipenv shell
```

5. Apply migrations:

```bash
python manage.py migrate
```

6. Run the development server:

```bash
python manage.py runserver
```
